2. Box Model
margin + border + padding + width / height = full size

3. Outline
* Wyjaśnić: Outline style
    outline-style: solid;
    outline-style: all;
    outline-style: top&bottom left&right;
    outline-style: top left&right bottom;
    outline-style: top right bottom left;
* Wyjaśnić: Outline width
    outline-width: 10px;
    outline-width: all;
    outline-width: top&bottom left&right;
    outline-width: top left&right bottom;
    outline-width: top right bottom left;
* Wyjaśnić: Outline color
    outline-color: red;
    outline-color: all;
    outline-color: top&bottom left&right;
    outline-color: top left&right bottom;
    outline-color: top right bottom left;
* Wyjaśnić: Outline sides
    outline-top-style: solid;
    outline-right-style: solid;
    outline-bottom-style: solid;
    outline-left-style: solid;
* Wyjaśnić: Outline shorthand
    outline: width style color;
* Wyjaśnić: Outline offset
     outline-offset: 15px;
